API Reference

This page acts as your verification API endpoint. Your Roblox game calls this URL to verify player IDs.

GET api.html?action=verify&id={REGISTRATION_ID}&key={API_KEY}

Verify Registration ID

Call this endpoint from your Roblox server-side Lua script using HttpService:GetAsync(). It returns JSON indicating whether the player's registration ID is valid.

ParameterTypeRequiredDescription
action string Required Must be verify
id string Required The player's Verification ID (from Player:GetJoinData().LaunchData)
key string Required Your API secret key (set in the Admin Panel → Configuration)

Example Response (valid ID):

{ "valid": true, "robloxUsername": "PlayerName", "registeredAt": 1720000000000 }

Example Response (invalid/revoked):

{ "valid": false }
⚠️ Production Note: This website uses localStorage, which means the API only works when the page is open in a browser on the same computer. For real-time verification across many players, you should host this as a proper web server (Node.js, Python Flask, etc.) with a database. The Roblox Lua script structure stays exactly the same.
